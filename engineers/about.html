How we learned CS

Here at Portland State University, we were taught from the ground up. We were taught using a linux system and doing all early coding from the command line. We primarily used C++ and vim with a heavy reliance on object-oreinted-design. This set a good foundation of the essential fundamentals for computer science. From there, we have all branched out in our studies, with things such as front and back end web development, database management, cybersecurity, and cryptography, and project management. Each of us brings a different state of mind that allows us to tackle our project effectively.
